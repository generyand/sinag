What You're Paying For

SINAG requires three services to run: a server to host the application, a database and file storage service to store all assessment data and uploaded documents, and an AI service for generating insights. Here is the cost for each:

Exchange rate used: PHP 58.00 = USD 1.00. Actual amounts may vary slightly depending on the prevailing rate. The AI service is currently on a free tier at no cost.

What Each Service Does

Server Hosting
Keeps the SINAG application running and accessible online 24/7. This is the computer in the cloud that serves the website to all users.
Database & File Storage
Stores all assessment records, user accounts, and uploaded documents (MOVs, logos, etc.). Currently maintains two databases: one for the live system and one for testing.
AI / Insights Service
Powers the intelligent features of SINAG such as automated analysis and recommendations. Currently free of charge.

Monthly Cost Expectations

The monthly cost may vary slightly from month to month due to normal fluctuations in usage. Based on current operations, you can expect:

  • Typical month: $65 – $74 (PHP 3,770 – PHP 4,292)
  • Most likely amount: around $71 (PHP 4,118)
  • Costs will only increase significantly if the system needs to handle substantially more users or data
